Drop Telugu wedding MP3 files here. The player expects these filenames:

  seetha-rama-kalyanam.mp3
  mangala-vadyam.mp3
  nadaswaram-mangalam.mp3
  sri-rama-jayarama.mp3

Each file in /public/music/ is served from the site root at /music/<filename>.
To change the track list or filenames, edit the PLAYLIST array in
src/routes/index.tsx.
